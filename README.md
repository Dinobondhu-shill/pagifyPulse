# PagifyPulse

## Client Side

### Project Name: PagifyPulse (Client)

### Project Overview
The client side of the PagifyPulse project is built using React, providing a user-friendly interface to browse products. The application allows users to search, sort, filter by price range, and paginate through the products.

### Features
- **Search**: Users can search for products by name.
- **Sorting**: Products can be sorted by date (Newest First, Oldest First) and price (High to Low, Low to High).
- **Filtering**: Users can filter products by brand, category, and price range.
- **Pagination**: Products are displayed with pagination, showing 10 products per page. Users can navigate through pages using page numbers and previous/next buttons.

### Technologies Used
- **React**: For building the user interface.
- **Tailwind CSS**: For styling the components.
- **Fetch API**: For making API requests to the server.
